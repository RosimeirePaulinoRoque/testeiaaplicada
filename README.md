Automação de Agendamentos - Rois Automações
Projeto desenvolvido para otimizar o fluxo de clínicas, utilizando n8n, Supabase e Telegram.

Tecnologias Utilizadas:
Banco de Dados: Supabase (PostgreSQL).

Automação: n8n (Low-code).

Notificações: Bot do Telegram.

Funcionalidades:
Verificação automática de agendamentos pendentes.

Consulta de dados de pacientes no banco de dados.

Envio de lembretes automáticos para evitar faltas.
